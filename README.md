# STUDENT-SCORESHEET
This Java console application simulates a student academic scoresheet generator.
It allows the user to enter:
School name
Teacher's name
The grade and year
Student names
Marks for 6 different subjects

Based on the input, the program calculates:
Total marks
Average
Grade (Aâ€“F)
Class totals and averages
Grade distribution statistics
The layout uses aligned columns with printf to simulate a real report card.

The project demonstrates:
Using the scanner class for user input
Using arrays to store data
Using for loops and appliying conditional statements to compute grades
Using System.out.printf() to create neatly aligned columns, header and footer sections, class statistics summary
